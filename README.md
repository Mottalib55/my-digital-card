# 75tools

Hub d'outils de productivité gratuits.

**URL**: https://75tools.fr

## Outils disponibles

- **MyCard** - Carte de visite digitale professionnelle
- **NetSalaire** - Calculateur de salaire net/brut

## Technologies

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase

## Développement local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
